# CUSTOM API
* Demonstration of how to build your own API using Node HTTP services.

## Features Available
* Do Basic Math.
  - Sums.
  - Squares.
  - Cubes.
  - Square Root.
  - Division.
* Get Date Info.
  - Current Time.
  - Current Date.
  - Days till X.
* Get Stock Info
  - Stock Information based on Symbol.
